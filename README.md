mozilo-2.0-bnAutumn
===================
Layout für moziloCMS 2.0 (https://github.com/mozilo/mozilo2.0)
Ein einfaches Layout mit Navigation auf der linken Seite und einer festen Breite von 960. Das Layout passt sich automatisch an kleinere Displays an.

Für Galerien wird das Plugin FotoGalerie empfohlen  ;) . (https://github.com/black-night/mozilo-2.0-FotoGalerie)
